# Каталог продуктовых направлений

Перечень технических направлений, вокруг которых [[organization|Alov Intelligence]]
планирует и сообщества, и open-source продукты. Это **намерения, а не выпущенные
продукты** — на 2026-06-26 ничего из этого ещё не существует как готовый продукт.
Разработка — снизу вверх, силами сообществ (см. [[open-source-first]]).

## Направления

| Направление | Заметки |
|-------------|---------|
| **Forecasting** | Прогнозирование (временные ряды, спрос и т.п.). |
| **LLMs** | Большие языковые модели. |
| **Computer vision** | Компьютерное зрение. |
| **Remote sensing** | Продукты дистанционного зондирования Земли. |
| **Network planning & optimization** | Планирование и оптимизация сетей. |
| **Supply and demand modeling** | Моделирование спроса и предложения. |
| **Geospatial solutions** | Геопространственные решения. |
| **SDI** | Spatial Data Infrastructure — инфраструктура пространственных данных. |
| **Geospatial ETL** | ETL для геоданных. |
| **Drones, self-driving, navigation** | Дроны, беспилотный транспорт, навигационные системы. |
| **Simulation, digital twins** | Симуляции и цифровые двойники. |

## Заметки

- Несколько направлений группируются вокруг **геопространства/гео-данных** (remote
  sensing, geospatial solutions, SDI, geospatial ETL, drones/navigation) — вероятный
  тематический кластер.
- Сообщества из стратегии (см. [[community-model]]) перечислены ýже продуктов: LLMs,
  computer vision, remote sensing, network optimization, supply/demand modeling,
  geospatial solutions. То есть продуктовый список **шире** списка первых сообществ.
- Приоритизация направлений («первые продукты») в источнике осталась открытым
  вопросом и здесь не фиксируется.

# raw/ — исходные источники

Курируемая коллекция **неизменяемых** источников для вики: заметки, статьи,
транскрипты встреч, экспорты, скриншоты, выгрузки данных.

## Правила

- Это **источник истины**. Агент только читает отсюда, никогда не правит.
- Изображения и вложения складывай в `raw/assets/`.
- Чтобы обработать источник: положи файл сюда и попроси агента «ингестнуть» его.
  Агент прочитает, обсудит выводы, напишет саммари в `wiki/sources/`, обновит
  связанные страницы, индекс и журнал (см. `CLAUDE.md` → Операции → Ingest).

Папки `raw/` и `wiki/` исключены из Jekyll-сборки и не публикуются на сайте.

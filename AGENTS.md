# AGENTS.md

This repository's agent instructions live in [`CLAUDE.md`](./CLAUDE.md).

`CLAUDE.md` is the single source of truth: it documents the Jekyll (Chirpy) site and
the LLM wiki under `wiki/` (and `raw/`), including conventions and the ingest / query /
lint workflows. Read it before doing any work in this repo.

Quick notes:

- **Wiki language is English only** — all pages in `wiki/` are written and maintained in
  English. Communication with the human is in Russian. See `CLAUDE.md` §7.
- **`raw/` is immutable** — read only, never edit. It is the source of truth for ingests.
- **Commit only when the human asks.**

For everything else, follow [`CLAUDE.md`](./CLAUDE.md).
